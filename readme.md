## 插件

koishi-plugin-kunzai 维护了一份 yunzai 插件子集（包括 yunzai 自身附带插件）并发布到 Koishi 插件市场中。你可以在插件市场中搜索 `kunzai` 来下载安装它们：

> 下列所有子集插件均继承自原插件许可证发布

- genshin：原神插件
- miao：喵喵 - 原神插件扩展
